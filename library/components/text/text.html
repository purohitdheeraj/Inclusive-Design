Usecase

  • Use Heading in a chronological order from large to small.
  • As a rule of thumb use one h1 per webpage.
  • by just using class you can apply this heading, paragraph style on block level, inline elements.
  • The heading components block level margins are 0 so you can use it directly without any modification.
