Usecase

  • you can use list with different styles like disc(default), circle, square with list-style-type class in this component.
  • There are multiple styles for ordered list like alphabets, romans
  • with class .list-display-inline and .list-style-none you can use it for nav links.
  • the width of notification-stacked can customised to fit the with of container.
